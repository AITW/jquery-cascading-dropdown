jQuery Cascading Dropdown Plugin Examples

Simple

In this example, Step 2 is dependent on Step 1 having a value. Step 3 is dependent on either one of Step 1 and Step 2 having a value.

$('#example1').cascadingDropdown({
	selectBoxes: [
		{
			selector: '.step1'
		},
		{
			selector: '.step2',
			requires: ['.step1']
		},
		{
			selector: '.step3',
			requires: ['.step1', '.step2']
		}
	]
});
